# Security Policy

## Supported Versions

We actively provide security patches and updates for the following versions:

| Version | Supported |
| ------- | --------- |
| v1.x.x  | ✅ Yes     |
| < v1.0  | ❌ No      |

---

## Reporting a Vulnerability

We take the security of CivicEcho and user data very seriously. If you discover a security vulnerability (such as exposed API credentials or client-side leaks), please do not report it in public issue trackers.

Instead, please report it privately following these steps:

1. Send a detailed report describing the vulnerability to the repository maintainer's contact email.
2. Include steps to reproduce the issue, along with any relevant proof-of-concept scripts or screenshots.
3. Allow us up to 48-72 hours to acknowledge the report and draft a patch before disclosing the vulnerability publicly.

Thank you for helping keep CivicEcho secure! 🛡️
